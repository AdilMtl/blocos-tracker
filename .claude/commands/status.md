# /status — Ver estado rápido do projeto

Mostra um resumo rápido sem carregar tudo.

## Ao receber este comando:

1. Verifique `git status` e `git log --oneline -5`
2. Conte as linhas do index.html
3. Liste alterações pendentes (se houver)
4. Apresente:

```
📊 STATUS

Linhas: XXXX
Branch: main
Último deploy: [mensagem] ([data])
Pendências: [lista ou "nenhuma"]
Sessão: [resumo do que foi feito nesta sessão, se houver]
```
